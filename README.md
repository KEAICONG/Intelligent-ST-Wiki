# Intelligent-ST-Wiki 
## 智能科学与技术专业学习指南
本项目旨在为智能科学与技术专业的同学提供学习建议、课程资料与实践经验，帮助大家更快建立专业认知，提升学习与实践能力。

## 前言：
智能科学与技术是一门交叉性较强的专业，课程内容涉及人工智能、计算机、自动化与信息处理等多个领域。面对较为广泛的知识体系，初学者难免感到迷茫。
本项目整理了部分专业课程复习资料与个人学习经验，希望能够为同学们的课程学习和能力提升提供参考。受个人能力与经验所限，部分内容可能存在不足，欢迎大家指正与补充。
