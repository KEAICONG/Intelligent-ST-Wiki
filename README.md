# Intelligent-ST-Wiki 
## 智能科学与技术专业学习指南

本项目旨在为智能科学与技术专业的同学提供学习建议、课程资料与实践经验，帮助大家更快建立专业认知，提升学习与实践能力。

## 写在前面

请大家善于使用 **AI 工具**。身处人工智能快速发展的时代，合理使用 AI 能够减少大量重复性工作，并辅助知识梳理、概念拆解与问题分析。

同时，也应重视**信息检索能力**的培养。无论是检索论文，还是查找各种文件、相关通知，都应尽量先自行搜索和核实，逐步提升信息素养，避免过度依赖他人直接提供答案。

遇到不熟悉的知识点时，可以在 [哔哩哔哩](https://www.bilibili.com/) 等平台查找相关讲解，往往会有意想不到的收获。

此外，请定期查阅 [杭州电子科技大学自动化学院官网](https://ac.hdu.edu.cn/main.htm) ，及时关注学院发布的重要通知。

## 前言：

智能科学与技术是一门交叉性较强的专业，课程内容涉及人工智能、计算机、自动化与信息处理等多个领域。面对较为广泛的知识体系，初学者难免感到迷茫。

本项目整理了部分专业课程复习资料与个人学习经验，希望能够为同学们的课程学习和能力提升提供参考。受个人能力与经验所限，部分内容可能存在不足，欢迎大家指正与补充。
